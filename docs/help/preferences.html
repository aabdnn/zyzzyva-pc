Preferences

Preferences can be configured by choosing the Edit menu item, then Preferences.

Auto Import

Zyzzyva can be configured to import a word list automatically each time it starts. To do so, select the Automatically import a word list on startup option. Click the Browse button to bring up a file browser, from which you can choose the word list file to be loaded.

Font Settings

To configure the font used by Zyzzyva, click the Choose button at the right of the Font selection area. This will bring up a font browser, from which you can choose the font to be used.

When you are done configuring your preferences, click the OK button to close the Preferences window. Any changes to the font setting will take effect immediately.
