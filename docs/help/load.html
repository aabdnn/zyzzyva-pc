Loading a Word List

In order for Zyzzyva to function properly, it must have a word list loaded. Load a word list by choosing the File menu item, then Open. Choose the word list file using the file browser that appears.

The word list file should contain a list of valid words, one per line. Each word can also be followed by a definition, separated from the word by one or more spaces. Here is an example of what a word list file might look like:

AA rough, cindery lava
AB abdominal muscle
AD advertisement

Instead of loading a word list manually each time you use Zyzzyva, you can configure Zyzzyva to load a word list automatically each time it starts. See Auto Import for more details.
