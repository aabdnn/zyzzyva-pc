Searching for Words

Choose File->New Search from the main menu to create a new Search tab. A Search tab can be used to find all acceptable words that match certain criteria. The criteria are specified in the top part of the Search tab, and the resulting list of words is displayed in the bottom area. Click the Search button to perform a search, and click the Reset button to reset the values in the search specification area.

Match Types

The main search criterion is a pattern of letters to be matched. This is specified in the Match section. A match can be specified as Pattern, Anagram, or Subanagram.

Pattern Match

The Pattern match type means that the search pattern must be matched exactly, in the same order the letters are specified. Special wildcard characters can be used to match any letters: the ? character means "match any single letter" and the * character means "match any number of letters". The * character also matches nothingness. Multiple wildcard characters can be used in a single pattern. Here are some examples of Pattern matches:

PatternMatches
ATEATE
AT?ATE, ATT
AT??ATAP, ATES, ATMA, ATOM, ATOP
T?PTAP, TIP, TOP, TUP
TO*PTOP, TOECAP, TOSSUP, TOUCHUP, TOYSHOP, TOWNSHIP
*WMCWM, SHAWM
?W*M?SSWAMIS, SWAMPS, TWASOMES, TWOSOMES, TWELVEMOS

Anagram Match

The Anagram match type means that the letters in the search pattern can be matched in any order. All the letters must be matched exactly the number of times they appear in the match pattern. The same special wildcard characters can be used as described in the Pattern Match section. Here are some examples of Anagram matches:

PatternMatches
ATEATE, EAT, ETA, TAE, TEA
AENORSTATONERS, SENATOR, TREASON
ETX?EXIT, NEXT, SEXT, TEXT, VEXT
PI??ZPIZZA, PRIZE, SPITZ, UNZIP, ZIPPY
*JBXJAMBEAUX

Note that with an Anagram match, using more than one * character has the same effect as using just one.

Subanagram Match

The Subanagram match type means that the letters in the search pattern can be matched in any order, and not all letters must be used in each matching word. The same special wildcard characters can be used as described in the Pattern Match section. Here are some examples of Subanagram matches:

PatternMatches
ATEAE, AT, ET, TA, ATE, EAT, ETA, TAE, TEA
LX?AL, AX, EL, EX, LA, LI, LO, OX, XI, XU, LAX, LEX, LOX, LUX

Note that with an Subanagram match, using a * character will cause the pattern to match every acceptable word.

Minimum and Maximum Length

A minimum and maximum length can be specified for the results of a search. To change these values, adjust the numbers in the Min Length and Max Length boxes. For example, if the minimum length is 4 and the maximum length is 6, then the search will only return results that are between four and six letters long. The minimum length cannot be greater than the maximum length.

Including and Excluding Letters

The input box labeled Must Include specifies letters that must be included in each word in the search results. This is most useful if you want to specify a Pattern match with wildcard characters, and want the wildcards to match certain letters in an unspecified order.

For example, to find all 7- and 8-letter words beginning with OVER and containing the letters A, L, and T in any order, do the following. Specify a Pattern match of OVER* with minimum length 7, maximum length 8, and Must Include ALT. This search will find the words OVERLATE, OVERSALT, and OVERTALK.

If a letter appears more than once in the Must Include box, the letter must appear at least that many times in each word in the search results.

The input box labeled Must Exclude specifies letters that must not be included in the search results. This is most useful for generating special word lists, such as the list of words containing the letter Q but not the letter U. To find such words, specify a Pattern match of * with Must Include Q and Must Exclude U. This search will find the 21 words containing Q but not U.

Letter Frequency

The area labeled Must Consist of specifies letters that must occur with a certain minimum frequency. This is useful for generating special word lists, such as the list of words containing at least 70% vowels. To find such words, set the value in the percentage box to 70, and put AEIOU in the input box.

Other Options

If the Use lower-case letters for wildcard matches checkbox is activated, any letters matched by a wildcard character will appear in lower case in the search results. For example, with this checkbox activated, an Anagram search of EXT? will result in EXiT, nEXT, sEXT, TEXt, vEXT rather than EXIT, NEXT, SEXT, TEXT, VEXT.
