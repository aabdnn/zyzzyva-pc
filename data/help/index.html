Welcome to Zyzzyva!

Zyzzyva is a program to help you study words and practice your anagramming skills. It allows you to search for words, anagrams, and definitions, as well as give yourself quizzes. Zyzzyva is particularly valuable for learning words for word games.

Help documentation for Zyzzyva can be found by browsing the Table of Contents. Please read the help documentation before getting started, so you are familiar with the capabilities of Zyzzyva.
